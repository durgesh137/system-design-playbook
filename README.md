# System Design Playbook

A minimal, practical guide to system design fundamentals and backend patterns. No fluff, no code—just concepts, checklists, and interview prompts to help you reason about scalable systems.

## Modules

1. **01-basics** – Core system design concepts and terminology
2. **02-everyday-backend-patterns** – Common backend operations with trade-offs and gotchas
3. **03-interview-questions** – 20 system design prompts with hints and trade-offs

## Goal

Keep it concise, actionable, and focused on real-world decision-making. Perfect for quick reviews, interviews, or architectural discussions.
