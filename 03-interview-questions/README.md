# 03 — Interview Questions (Curated)

Summary
- 20 focused system-design prompts grouped by difficulty, with 3 quick hints and a 1-sentence note about trade-offs.

Example prompts
- Easy: Design a URL shortener — hints: unique id generation, collision handling, redirect latency; trade-off: simplicity vs geo-replication.
- Medium: Design a file upload service for large files — hints: direct-to-cloud, resumable uploads, virus scanning; trade-off: complexity vs cost.
- Hard: Design a globally-consistent shopping cart — hints: eventual vs strong consistency, conflict resolution, latency; trade-off: UX vs correctness.

How to use
- Use each prompt in interviews or to generate a problem-card for study.
